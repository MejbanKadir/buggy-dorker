## ğŸ“– How to Use

### Basic Usage
1. **Select a Dorking Category**: Choose from the available tabs, such as Blind XSS, Files Dorker, or URL Matcher.
2. **Enter Your Target**: Input the domain or keyword you want to investigate.
3. **Generate and Execute**: Click the search button to generate a Google dork and open it in a new tab.

### Advanced Usage
- **Customize Dorks**: Use the URL Matcher to build custom queries with operators like `inurl`, `intext`, and `intitle`.
- **File Dorking**: Search for specific file types and filter by date to find exposed documents.
- **GitHub Dorking**: Hunt for sensitive information in public code repositories.
- **Shodan Integration**: Construct Shodan queries to discover internet-connected devices and services.
