Select one or more file types, optionally set a date range, then launch the Google dork.

Pick one or more file types to hunt for exposed documents on the target domain.

Add keywords for each operator to build advanced queries. Leave any field blank to skip that operator.

Mix operators to narrow results. Example: `site:example.com & inurl:login & allintext:"reset password"`.

Google Dork Operators Cheat Sheet

allintext
Finds pages where all specified keywords appear in the body text.
intext
Locates pages that include any of the keywords within the body content.
inurl
Searches for URLs containing at least one of the provided keywords.
allinurl
Returns URLs that include every keyword listed in the query.
intitle
Targets pages whose titles contain any of the given keywords.
allintitle
Matches pages with titles that include all keywords together.
link
Finds pages that link externally to the specified URL or keyword.

Select presets to search GitHub code for exposed references to the target domain. We include the domain automatically.

Each preset adds secret-hunting keywords so you can quickly scan GitHub results that mention the domain. Example: "example.com" example.com path:**/.env "SECRET_KEY" "APP_KEY" "DATABASE_URL".

Build custom Shodan queries using various filters and operators. Leave any field blank to skip that filter. The domain will be automatically added as a hostname filter.

Mix filters to create precise queries. Example: `hostname:example.com product:"nginx" port:443 country:"US"`.

Shodan Search Filters Cheat Sheet

product
Search for specific software/services (e.g., nginx, Apache, MySQL).
port
Filter by specific ports (e.g., 80, 443, 22, 3389).
country
Filter by country code (e.g., US, GB, DE, CN).
city
Filter by city name (e.g., London, Paris, Tokyo).
org
Filter by organization name (e.g., Google, Amazon, Microsoft).
os
Filter by operating system (e.g., Windows, Linux, Ubuntu).
vuln
Search for specific vulnerabilities (e.g., CVE-2021-44228).
title
Search in HTTP title tags (e.g., login, admin, dashboard).
html
Search in HTML content (e.g., password, form, login).
net
Search within IP ranges (e.g., 192.168.1.0/24).
ssl
SSL certificate filters (e.g., expired, self-signed).
version
Filter by software version (e.g., 2.4.41, 1.18.0).
